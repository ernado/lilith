# Форматирование сообщений (Markdown)

Сообщения отправляются в Telegram через ограниченный парсер Markdown.
Поддерживается только то, что перечислено ниже. Всё остальное Telegram
покажет как обычный текст вместе с символами разметки (например, `#`, `|`,
`-`), поэтому НЕ используй неподдерживаемые конструкции.

## Что поддерживается

```
*курсив*  или  _курсив_
**жирный**  или  __жирный__
~~зачёркнутый~~
`моноширинный код`
[текст ссылки](https://t.me/)
[упоминание](tg://user?id=123456789)
> цитата
```

Блок кода с указанием языка:

````
```python
print("код")
```
````

## Что НЕ поддерживается (НЕ используй)

- Заголовки (`#`, `##`, …) — символы `#` останутся в тексте.
- Списки (`-`, `*`, `1.`) — маркеры останутся в тексте.
- Таблицы.
- Горизонтальные разделители (`---`).
- Спойлеры (`||...||`).
- Выделение (`==...==`).
- Формулы (`$...$`, `$$...$$`).
- Подчёркивание.
- HTML-теги.
- Картинки по ссылке (`![](url)`).

## Как оформлять структуру

Вместо заголовков и списков используй обычный текст, переносы строк и, при
необходимости, **жирный** для выделения. Не имитируй списки символами `-` или
`1.` — пиши пунктами через переносы строк.
